Glossary

Shared words used by docs, code, issues, and agents.

Home Architecture Phase 0 Markdown source

Chunk

A fixed-size block of audio passed through the voice conversion pipeline.

Provider

An ONNX Runtime execution backend, such as CUDA, DirectML, CoreML, OpenVINO, or CPU.

Underrun

The output device requests audio but the runtime has too few frames ready.

ADR

A numbered record of one architectural decision and its consequences.
